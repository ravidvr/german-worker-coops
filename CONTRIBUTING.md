# Contributing

## Quickest way: Open an Issue

Click this button to submit a new entry:

👉 **[➕ Add a worker-owned business](https://github.com/ravidvr/german-worker-owned-businesses/issues/new?template=add-entry.yml)**

Fill in the form — that's it. A maintainer will review and add it to the database.

## Or submit via Pull Request

1. Fork the repo
2. Edit `data/worker_coops_germany.json`
3. Add your entry following the existing schema (include `lat`/`lng` coordinates if possible)
4. Submit a PR with a brief verification note

## What qualifies

- 51%+ of employees are member-owners with voting rights → `worker_majority`
- Workers have elevated decision-making power through governance structures → `governance_based`
- Workers have elevated decision-making power by statute → `statute_based`
- In the process of establishing worker ownership → `in_founding`

## Verification

Please include a source or note explaining how you verified the business is worker-owned. This can be as simple as a link to their "About" page or a note about personal knowledge.
